Introduction to Depth tesing in OpenGLES.

== Description ==
Depth Testing helps in rendering the objects based on the depth distance from viewing position rather than the rendering order. This sample application demonstrate the capability of OpenGLES to perform depth testing.  Tap on screen to enable and disable depth testing. 

== Installation ==
Importing sample source in Android platform:-
1. Go to New-> Project -> Android Project from Existing Code. Click Next
2. Specify the path of the project. Click Finish
Note: The path of the project should be the directory name which contains Android.xml.
