Introduction to Rendering Primitives in OpenGLES.

== Description ==
This sample application demonstrate various type of rendering primitives available in OpenGLES. Tap on screen to view the models in different type of primitives. 


== Installation ==
Open the project by double clicking the .xcodeproj file.
