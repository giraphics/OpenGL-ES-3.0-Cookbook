Introduction to Transformation operation in OpenGLES.

== Description ==
This sample application demonstrate orthographic and perspective projection system in 3D graphics. In order to switch between the two project system tap on the screen. 


== Installation ==
Importing sample source in Android platform:-
1. Go to New-> Project -> Android Project from Existing Code. Click Next
2. Specify the path of the project. Click Finish
Note: The path of the project should be the directory name which contains Android.xml.
