Introduction to Transformation operation in OpenGLES.

== Description ==
This sample application demonstrate orthographic and perspective projection system in 3D graphics. In order to switch between the two project system tap on the screen. 


== Installation ==
Open the project by double clicking the .xcodeproj file.
