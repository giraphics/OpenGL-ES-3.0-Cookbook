Touch Events in OpenGLES using GLPI Framework.

== Description ==
This recipe will demonstrate the use of touch events in iOS and Android OpenGLES application. Using this recipe you will be able to distinguish b/w single and double tap, events like press, move and release.

== Installation ==
Importing sample source in Android platform:-
1. Go to New-> Project -> Android Project from Existing Code. Click Next
2. Specify the path of the project. Click Finish
Note: The path of the project should be the directory name which contains Android.xml.
