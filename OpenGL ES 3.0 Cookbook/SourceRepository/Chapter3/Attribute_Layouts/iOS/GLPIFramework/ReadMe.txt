Introduction to Storage and Layout Qualifier.

== Description ==
A Storage Qualifier are special keywords that specifies the storage or the behaviour of a global or local variable. They are used in shader programming, they enable the communication bridge between the application and shaders. They are also used to share information from one shader stage to another.
