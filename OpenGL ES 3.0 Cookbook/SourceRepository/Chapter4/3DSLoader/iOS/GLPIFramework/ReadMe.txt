Parsing and Rendering of 3DS mesh model.

== Description ==
This application build a parser to read the 3DS mesh files. It will also display the mesh model using triangle, line and point primitive. Tap the screen to see the renderig of mesh with point, line and triangle.

== Installation ==
Open the project by double clicking the .xcodeproj file.
