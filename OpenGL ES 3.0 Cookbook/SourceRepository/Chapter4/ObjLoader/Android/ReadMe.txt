Parsing and Rendering of OBJ mesh model.

== Description ==
This application build a parser to read the Obj mesh files. It will also display the mesh model using triangle, line and point primitive. Tap the screen to see the renderig of mesh with point, line and triangle.

== Installation ==
Importing sample source in Android platform:-
1. Go to New-> Project -> Android Project from Existing Code. Click Next
2. Specify the path of the project. Click Finish
Note: The path of the project should be the directory name which contains Android.xml.
